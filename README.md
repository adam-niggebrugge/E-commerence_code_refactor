## Installation

```

npm init

npm install mysql2

npm install express

npm install sequelize

```

must set up a .env file based on MySQL installed settings. It will look like below with <> replaced based on installed setting.

```

DB_USER='<user_name_here>'
DB_PW='<password_here>'
DB_NAME='ecommerce_db'

```
